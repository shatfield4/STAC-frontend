## Installation

```
git clone git@github.com/GrandTheftBacon/webapp-grandtheftbacon-game
```

```
cd STAC-frontend/
```

```
npm install
```

> Run **Locally**

```
npm run dev
```

> Generate **Static** Pages

```
npm run build:static
```

```
cd dist/
```

```
npx serve
```
